# Nook

Nook is your online book corner. Built to make finding books, storing notes and rating books easy and accessible and convinient. 

## üöÄ Features
- **User authentification and profiles** - User are able to create secure profiles.
- **Dashboard** - Users can view their saved books, book ratings, in progress reads, and their sticky notes.
- **Sticky Notes** - Users are able to add comments and notes about the books the have read.

## üîß Installation

1. **Clone the repo**

   ```bash
   git clone https://github.com/Lee-nkoana/nook.git
   cd nook
   ```

2. ## Create & activate a virtual environment

   ```bash
   python -m venv venv
   source venv/bin/activate  # Mac/Linux
   venv\Scripts\activate     # Windows
   ```

3. ## Install dependencies

   ```bash
   pip install -r requirements.txt
   ```

4. ## How to run the app
   ```bash
   python manage.py runserver
   ```
